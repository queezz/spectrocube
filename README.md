# spectrocube

Lightweight standard for calibrated spectroscopic datasets

## Quickstart

```bash
python -m pip install -e ".[dev]"
pytest
```

## Docs

```bash
python -m pip install -e ".[docs]"
mkdocs serve
```


## Virtual Environment

Create virtual environment

Linux / macOS:
```
python3 -m venv ~/.venvs/spectrocube
```

Windows PowerShell:
```
python -m venv "$env:USERPROFILE/.venvs/spectrocube"
```

Activate virtual environment

Linux / macOS:
```
source ~/.venvs/spectrocube/bin/activate
```

Windows PowerShell:
```
& "$env:USERPROFILE/.venvs/spectrocube/Scripts/Activate.ps1"
```
